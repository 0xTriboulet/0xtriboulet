# Steve S. (0xTriboulet)

Security Researcher.

My public repositories consist of templates, tooling, and code collections that support both offensive security development and systems engineering. 


## Featured Projects

- **[`rdll-rs`](https://github.com/0xTriboulet/rdll-rs)** — A reflective DLL development template written in Rust, supporting DLL and executable builds. 
- **[`InlineExecuteEx`](https://github.com/0xTriboulet/InlineExecuteEx)** — A Beacon Object File loader and execution extension built in C++. 
- **[`scepter-rs`](https://github.com/0xTriboulet/scepter-rs)** — Cross-architecture/platform support utilities for Cobalt Strike in Rust. 
- **[`emerald_template`](https://github.com/0xTriboulet/emerald_template)** — A CMake template for developing and debugging reflective DLL loaders. 
- **[`ai-postex`](https://github.com/0xTriboulet/ai-postex)** — Proof-of-concept AI-augmented post-exploitation DLLs. 

Other interesting stuff includes machine-learning-enabled shellcode loaders with identity verification and third-party agent extensions for C2 frameworks.
